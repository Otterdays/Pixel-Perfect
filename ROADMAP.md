# Pixel Perfect Roadmap

## Guiding Goals
- [ ] Keep UX fast, clear, and forgiving (no accidental loss)
- [ ] Make common tasks 1–2 clicks/keys
- [ ] Preserve retro aesthetic while improving clarity
- [ ] Ship playable, stable builds over perfection

---

## Phase 1: Quick Wins (Easy–Medium)
- [ ] Flip Vertical (mirror selection on Y-axis)
- [ ] Recent Colors Panel (last 10–20 colors)
- [ ] Color Swap Hotkey (`X`) for primary/secondary
- [ ] Temporary Eyedropper (hold Alt to sample)
- [ ] Zoom to Cursor (Ctrl+wheel focuses on pointer)
- [ ] Fit Canvas / 100% View buttons
- [ ] Export Presets (save PNG/GIF/spritesheet options)
- [ ] Batch Export (multiple frames/layers)
- [ ] Repeating Edges Preview (tile seam checker)
- [ ] Grid Line Options (thickness + color)
- [ ] Quick Export button (reuse last export settings)
- [ ] Per‑Tool Size Memory (brush/eraser/spray keep sizes)
- [ ] Color Swap Hotkey (`X`) with UI hint
- [ ] Draw with Secondary (hold Alt while drawing)
- [ ] Hover Color Readout (RGBA + palette index)
- [ ] Recent Export Paths dropdown
- [ ] Autosave Status Indicator (Saved/Unsaved)
- [ ] Undo History Tooltip (last action)
- [ ] Nudge Selection (arrows, Shift=5px)
