Frequently Asked Questions

Is Pixel Perfect free?

The source code is available on GitHub. Download the standalone executable and use it for your projects.

Do I need to install Python?

No! The standalone .exe includes everything needed. Just download, extract, and run.

Can I use my own color palettes?

Yes! Drop any 16-color JSON palette file into the assets/palettes folder and it appears automatically.

What file formats are supported?

Import/export PNG, export GIF animations and sprite sheets. Save projects as .pixpf files.

What canvas sizes are available?

Presets: 8×8, 16×16, 32×32, 16×32, 32×64, 64×64 pixels. Custom sizes also supported.

Is there Mac or Linux support?

Currently Windows only. Mac/Linux users can run from source with Python 3.13+.
